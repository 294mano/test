# 訂單查詢系統

這是一個簡單的訂單查詢系統，允許用戶通過電子郵件查詢訂單狀態。

## 功能

- 通過電子郵件查詢訂單狀態
- 提交問題功能
- 即時顯示查詢結果

## 使用方法

1. 輸入您的電子郵件地址
2. 點擊"送出"按鈕查詢訂單狀態
3. 如果需要提出問題，可以在下方的問題框中輸入

## 技術說明

- 使用 HTML5, CSS3, JavaScript
- Bootstrap 5 用於 UI 設計
- Google Apps Script 作為後端 API
- JSONP 用於跨域請求
